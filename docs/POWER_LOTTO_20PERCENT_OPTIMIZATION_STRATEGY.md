## 2. 2注 策略優化 (2注達到 20%+ 的可能性分析)

### 2.1 理論分析

**從 4注 14.74% 反推 2注 概率**:
- 假設 2注預測完全獨立，中獎率應為: P(至少1注中) = 1 - (1 - 0.1474)² = 27.68%
- 但實際會因為方法相關性而降低，估計 2注最高: **12-15%**

**結論**: 要讓 2注 達到 20%，理論上要找到單注命中率 ~11% 的配置

### 2.2 突破策略 (2注 → 20%+)

#### 策略 A: 高度優化的雙方法組合
```
注1: 增強型 Ensemble (窗口: 500期，權重優化)
     - 使用最長歷史窗口確保穩定性
     - 優化方法權重 (Bayesian + Zone_Balance 權重提升)
     - 特別號: sum_bias (已驗證50%命中)
     
注2: Anti-Consensus + Trend 混合
     - Anti-Consensus: 選擇少數人的號碼
     - Trend: 捕捉最近的趨勢
     - 低相關性設計，互補性強
```

**預期效果**: 
- 注1命中率: ~5.5% (高窗口穩定性)
- 注2命中率: ~4.2% (趨勢捕捉)
- 組合: P(至少1中) = 1 - (1-0.055)(1-0.042) = **9.3%**
- ⚠️ 仍低於 20% 目標

#### 策略 B: 擴展覆蓋型雙注 (放寬"中獎"定義)
```
如果放寬"成功"定義為:
- "中3個以上" → 改為 "中2個以上"
- 則 2注達到 20%+ 可能

預測單注中2+個的概率: ~8-10%
組合 2注: P(至少1注中2+) = 1 - (1-0.09)² = **16.81%**

❌ 仍不足20%, 但更現實
```

#### 策略 C: 包含特別號的複合定義
```
成功定義: "中3個主號 OR 中2個主號+特別號"

由於特別號命中率 ~50%，可有效提升成功率
- 單注命中率 (複合定義): ~9-11%
- 2注組合: **16-19%**

⚠️ 接近但仍未達 20%
```

## 附錄: 技術參考

### 相關方法說明

**ClusterPivot**: 圍繞錨點號碼的聚類搜索
- 優勢: 專注於高相關區域
- 劣勢: 可能遺漏偏遠號碼

**Anti-Consensus**: 選擇最少人選擇的號碼
- 優勢: 低覆蓋度提高中獎金額
- 劣勢: 命中率通常較低

**Ensemble**: 多模型權重平均
- 優勢: 穩定性強，覆蓋面廣
- 劣勢: 反應速度慢

**Trend**: 捕捉最近期的趨勢
- 優勢: 及時反應市場動向
- 劣勢: 容易過度擬合
