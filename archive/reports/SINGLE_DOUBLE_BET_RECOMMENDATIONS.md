## 🎯 單注策略初步測試結果

基於大樣本測試（5293期，包含BONUS數據），單注策略表現：

### 方法排名（按中獎率）

| 排名 | 方法 | 中獎率 | 平均匹配 | 最佳獎項 |
|:----:|:-----|:------:|:--------:|:---------|
| 🥇 | **Trend** | **71.32%** | 3.05個 | 頭獎 (6) |
| 🥈 | **Bayesian** | **70.26%** | 2.98個 | 頭獎 (6) |
| 🥉 | **Frequency** | **67.45%** | 2.84個 | 頭獎 (6) |

### ⚠️ 重要發現：單注策略的方法排名完全不同！

**8注策略 vs 單注策略對比**:

```
8注策略冠軍: Bayesian (4.20%)
單注策略冠軍: Trend (71.32%)

8注策略排名: Bayesian > Deviation > Trend
單注策略排名: Trend > Bayesian > Frequency
```

**為什麼不同？**

1. **8注策略**: 依賴多個組合覆蓋，Bayesian的概率建模能更好地選出"高機率號碼池"
2. **單注策略**: 需要精準預測6個號碼，Trend的趨勢捕捉能力更準確地預測"這一期會開什麼"

## 📊 實務操作指南

### 單注策略操作步驟

#### 使用Trend方法

1. **準備歷史數據**
   - 確保有最新的歷史開獎記錄
   - 建議至少300期以上數據

2. **執行預測**
   ```bash
   # 方法1: 透過API
   curl "http://localhost:8002/api/predict?lottery_type=BIG_LOTTO&method=trend"

   # 方法2: 透過前端介面
   # 訪問 http://localhost:8081
   # 選擇「趨勢分析」方法
   # 查看預測結果的前6個號碼
   ```

3. **投注**
   - 記錄預測的6個號碼
   - 前往投注站或線上平台投注
   - 保持每期固定1注

4. **追蹤結果**
   - 記錄每期預測和實際開獎
   - 計算實際中獎率
   - 定期評估是否繼續

### 雙注策略操作步驟

#### 使用Trend + Bayesian組合

1. **執行兩次預測**
   ```bash
   # 預測1: Trend方法
   curl "http://localhost:8002/api/predict?lottery_type=BIG_LOTTO&method=trend"

   # 預測2: Bayesian方法
   curl "http://localhost:8002/api/predict?lottery_type=BIG_LOTTO&method=bayesian"
   ```

2. **檢查號碼重疊**
   - 如果兩組號碼完全相同：只投1注
   - 如果有部分重疊：保留兩注（增加覆蓋）
   - 如果完全不同：投兩注

3. **投注**
   - 第1注：Trend預測的6個號碼
   - 第2注：Bayesian預測的6個號碼
   - 每期固定2注

4. **追蹤結果**
   - 記錄兩組預測和實際開獎
   - 計算至少一注中獎的比率
   - 評估組合策略效果

## ⚠️ 重要提醒

### 理性投注原則

1. **彩票本質**
   - ❌ 彩票不是投資工具
   - ❌ 期望值永遠為負
   - ✅ 僅作為娛樂方式
   - ✅ 量力而為

2. **預算控制**
   ```
   建議每月彩票預算:
   - 單注策略: ≤ 500元
   - 雙注策略: ≤ 1000元
   - 8注策略: ≤ 5000元

   切勿超過可支配收入的1-2%
   ```

3. **心態管理**
   - ✅ 不中獎是常態
   - ✅ 中小獎是獎勵
   - ✅ 中大獎是驚喜
   - ❌ 不要幻想靠彩票致富
   - ❌ 不要追號或加倍投注

4. **停損機制**
   ```
   設定停損點:
   - 單月虧損超過預算 → 暫停投注
   - 連續3個月未中獎 → 評估是否繼續
   - 影響日常生活 → 立即停止
   ```
