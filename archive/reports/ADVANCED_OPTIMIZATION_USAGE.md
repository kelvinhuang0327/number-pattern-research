# 🚀 進階自動優化功能使用指南

## 📖 概述

進階自動優化系統已成功整合到前端 UI，提供兩種強大的優化方法來大幅提升預測成功率（預期從 3.61% 提升至 8-15%）。

## ✨ 功能特性

### 1. 🎯 多階段優化 (Multi-Stage Optimization)

**原理**: 通過三個階段逐步逼近最優解
- **階段 1**: 粗調 (50代, 30% 變異率) - 快速探索解空間
- **階段 2**: 精調 (100代, 15% 變異率) - 深度優化參數
- **階段 3**: 微調 (50代, 5% 變異率) - 精確調整

**預期效果**:
- ⏱️ 耗時: 10-15 分鐘
- 📈 提升幅度: 50-100% 適應度
- 🎯 目標成功率: 8-12%

### 2. 🔍 自適應窗口優化 (Adaptive Window Optimization)

**原理**: 自動測試不同數據窗口大小，找出最佳訓練數據量
- 測試窗口: 100期, 200期, 300期, 500期, 全部數據
- 自動選擇最佳表現的窗口大小

**預期效果**:
- ⏱️ 耗時: 5-8 分鐘
- 📈 提升幅度: 20-30% 適應度
- 🎯 目標成功率: 6-10%

## 🎮 使用步驟

### 步驟 1: 準備數據

1. 確保已上傳至少 **300 期以上**的彩票歷史數據
2. 選擇要優化的彩票類型（如大樂透、威力彩等）

### 步驟 2: 啟動進階優化

1. 進入「🤖 AI 自動學習」頁面
2. 滾動至「🚀 進階優化系統」區域
3. 選擇優化方法：
   - 點擊「執行多階段優化」- 追求最高準確率
   - 點擊「執行自適應窗口優化」- 快速找出最佳數據窗口

### 步驟 3: 監控進度

- 優化過程在後台執行，不會阻塞 UI
- 系統會顯示通知提示優化狀態
- 可以在優化期間繼續使用其他功能

### 步驟 4: 查看結果

優化完成後：
- 最佳配置會自動保存
- 點擊「載入最佳配置」查看優化參數
- 查看「優化歷史」表格中的最新記錄

## ⚠️ 注意事項

### 數據要求
- ❌ 數據不足（<100期）會無法執行
- ⚠️ 數據量 100-300期 可以執行但效果可能有限
- ✅ 數據量 300期以上 效果最佳

### 使用建議
1. **首次使用**: 先嘗試「自適應窗口優化」（較快）
2. **追求極致**: 使用「多階段優化」獲得最佳效果
3. **定期優化**: 每週或每50期新數據後重新優化
4. **多次對比**: 執行多次取平均值，減少隨機性影響

### 常見問題

**Q: 為什麼優化時間這麼長？**
A: 進階優化需要測試大量參數組合（多階段優化共200代遺傳算法），以找到最優解。這是提升準確率的必要代價。

**Q: 可以同時執行兩種優化嗎？**
A: 不建議。為避免資源競爭，請等待一個優化完成後再執行另一個。

**Q: 優化會影響其他功能使用嗎？**
A: 不會。優化在後台執行，前端 UI 保持響應，可以繼續瀏覽和預測。

**Q: 如何知道優化是否成功？**
A: 查看以下指標：
- 適應度是否提升（目標 >8%）
- 實際預測準確率是否提高
- 優化歷史表格中的最新記錄

## 🆘 問題排查

### 問題 1: 點擊按鈕無反應
**原因**: 未選擇彩票類型
**解決**: 先到「數據上傳」頁面選擇彩票類型

### 問題 2: 提示數據不足
**原因**: 歷史數據少於100期
**解決**: 上傳更多歷史數據（建議300期以上）

### 問題 3: 優化失敗
**原因**: 後端 API 未運行或崩潰
**解決**:
```bash
cd lottery-api
python3 app.py
```

### 問題 4: 離線模式提示
**原因**: 前端無法連接到後端
**解決**: 檢查後端是否在 5001 端口運行
