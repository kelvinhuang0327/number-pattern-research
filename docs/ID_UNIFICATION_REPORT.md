## 🎯 解決的問題

### 1. **數據匹配錯誤** ✅
- 前端上傳的數據使用 `POWER_BALL`
- 後端期望 `POWER_LOTTO`
- **結果**: 數據無法正確儲存或查詢

### 2. **預測功能異常** ✅
- 前端請求 `DAILY_CASH_539` 的預測
- 後端找不到對應的規則配置
- **結果**: 預測失敗或使用錯誤的規則

### 3. **統計數據不準確** ✅
- 前後端統計不同ID的數據
- **結果**: 數字不一致，混淆使用者

## 📝 技術細節

### 命名規則
統一採用以下命名規則：
- **主要遊戲**: `GAME_NAME` (如 `BIG_LOTTO`, `POWER_LOTTO`)
- **星彩系列**: `N_STAR` (如 `3_STAR`, `4_STAR`)
- **樂合彩系列**: `NN_LOTTO` (如 `39_LOTTO`, `49_LOTTO`)
- **特殊版本**: `GAME_MODIFIER` (如 `BIG_LOTTO_BONUS`)

### 特殊標記新增
為樂合彩系列添加了 `dependsOn` 屬性：
- `39_LOTTO` → dependsOn: `DAILY_539`
- `49_LOTTO` → dependsOn: `BIG_LOTTO`
- `38_LOTTO` → dependsOn: `POWER_LOTTO`
