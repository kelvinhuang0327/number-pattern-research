## 🎯 蒙地卡罗策略设计

### 核心理念

使用概率加权采样 + 多目标优化评分系统

```
蒙地卡罗模拟流程:
  1. 从历史数据学习号码概率分布
  2. 预测目标奇偶比（1或2个奇数）
  3. 生成5000组候选组合（概率加权采样）
  4. 对每组进行综合评分
  5. 选择得分最高的组合
```

### 评分系统（5个维度）

| 维度 | 权重 | 说明 |
|------|------|------|
| **奇偶比匹配度** | 40% | 是否符合1奇4偶或2奇3偶 |
| 号码频率得分 | 20% | 基于历史出现概率 |
| 反共识得分 | 20% | 避开生日范围、连号等 |
| 差异度得分 | 10% | 与最近3期的差异 |
| 间隔合理性 | 10% | 号码分布均匀度 |

**总分计算**:
```
总分 = 奇偶比匹配(40%) + 频率(20%) + 反共识(20%) + 差异度(10%) + 间隔(10%)
```

1. **概率分布学习**
   ```python
   # 基于最近100期，使用Laplace平滑
   probs[num] = (count + 1) / (total + max_num)
   ```

2. **奇偶比预测**
   ```python
   # 统计最近50期，选择1奇或2奇
   target = 1 if count_1_odd >= count_2_odd else 2
   ```

3. **加权采样**
   ```python
   # 分离奇偶池，按概率采样
   selected_odds = np.random.choice(odds, size=target, p=odd_probs)
   selected_evens = np.random.choice(evens, size=5-target, p=even_probs)
   ```

4. **多目标评分**
   ```python
   total_score = (
       odd_even_score * 0.40 +

   )
   ```

5. **模拟优化**
