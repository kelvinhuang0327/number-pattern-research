## 🔍 驗證方法說明

### 滾動式回測 (Rolling Backtest)

**標準流程**:
```
FOR each period P in [1 to N]:
    history = all_draws[P+1:]         // 該期 **之後** 的全部歷史
    prediction = predict(history)     // 基於歷史預測
    actual = all_draws[P]             // 該期實際開獎
    
    IF matches(prediction, actual):
        hits += 1
    total += 1

win_rate = hits / total
```

**防洩漏保障**:
- ✅ 預測時刻: 該期開獎 **前**
- ✅ 使用數據: 該期開獎 **後** 的歷史
- ✅ 時序檢查: 無逆向時間訪問
- ✅ 參數固定: 預測時不調整參數
