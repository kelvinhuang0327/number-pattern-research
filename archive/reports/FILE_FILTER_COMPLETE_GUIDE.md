# 大樂透檔案過濾完整指南

## 📋 概述

本系統現在**僅支援上傳「大樂透」檔案**，會自動過濾並忽略所有其他彩券類型的檔案。

---

## 🎯 過濾規則

### ✅ 允許上傳的檔案

檔案名稱必須同時滿足：
1. **包含**「大樂透」關鍵字
2. **不包含**任何排除關鍵字

#### 範例：
- ✅ `大樂透_2024.csv`
- ✅ `大樂透開獎項_113年.csv`
- ✅ `Taiwan_Lotto_大樂透_2023.csv`
- ✅ `大樂透歷史數據.csv`

### ❌ 自動忽略的檔案

包含以下任一關鍵字的檔案會被自動忽略：

| 關鍵字 | 說明 |
|--------|------|
| `加開` | 大樂透加開獎項 |
| `賓果` | 賓果賓果 |
| `bingo` | Bingo (英文) |
| `威力彩` | 威力彩 |
| `今彩` | 今彩539 |
| `539` | 今彩539 |
| `38樂合彩` | 38樂合彩 |
| `49樂合彩` | 49樂合彩 |
| `雙贏彩` | 雙贏彩 |
| `三星彩` | 三星彩 |
| `四星彩` | 四星彩 |

#### 範例：
- ❌ `大樂透加開獎項_2024.csv` (包含「加開」)
- ❌ `威力彩_2024.csv` (不包含「大樂透」)
- ❌ `今彩539_2024.csv` (不包含「大樂透」)
- ❌ `賓果賓果_2024.csv` (不包含「大樂透」)
- ❌ `lottery_data.csv` (不包含「大樂透」)

## 🔄 上傳流程

### 單檔上傳

1. 點擊「選擇檔案」按鈕
2. 選擇 CSV 檔案
3. 系統檢查檔案名稱
   - ✅ 如果是大樂透檔案 → 開始上傳
   - ❌ 如果不是大樂透檔案 → 顯示警告並拒絕

**警告訊息範例**：
```

```

1. 點擊「選擇多個檔案」按鈕
2. 選擇包含多個 CSV 檔案的資料夾
3. **系統自動過濾**：
   - 分類：大樂透檔案 vs 其他檔案
   - 顯示過濾結果通知
   - **僅上傳大樂透檔案**

## 💻 技術實現

### 1. 過濾函數 ([src/core/App.js:77-112](src/core/App.js#L77-L112))

```javascript
shouldIgnoreFile(filename) {
    const lowerFilename = filename.toLowerCase();

    // ✅ 只允許「大樂透」的檔案
    const allowKeywords = ['大樂透'];

    // ❌ 排除關鍵字
    const rejectKeywords = [
        '加開', '賓果', 'bingo', '威力彩', '今彩', '539',
        '38樂合彩', '49樂合彩', '雙贏彩', '三星彩', '四星彩'
    ];

    const hasRejectKeyword = rejectKeywords.some(keyword =>
        lowerFilename.includes(keyword)
    );

    const hasAllowKeyword = allowKeywords.some(keyword =>
        lowerFilename.includes(keyword)
    );

    // 只有包含「大樂透」且不包含排除關鍵字的檔案才允許上傳
    return !hasAllowKeyword || hasRejectKeyword;
}
```

### 3. 多檔上傳處理 ([src/core/App.js:446-483](src/core/App.js#L446-L483))

```javascript
async handleMultipleFileUpload(files) {
    // 🔍 過濾檔案：只保留「大樂透」檔案
    const filteredFiles = [];
    const ignoredFiles = [];

    for (let i = 0; i < files.length; i++) {
        if (this.shouldIgnoreFile(files[i].name)) {
            ignoredFiles.push(files[i].name);  // 記錄忽略的檔案
        } else {
            filteredFiles.push(files[i]);       // 保留大樂透檔案
        }
    }

    // 顯示過濾結果
    if (ignoredFiles.length > 0) {
        this.uiManager.showNotification(
            `⚠️ 已自動過濾非大樂透檔案\n\n` +
            `📋 將載入: ${filteredFiles.length} 個大樂透檔案\n` +
            `🚫 已忽略: ${ignoredFiles.length} 個檔案`,
            'warning'
        );
    }

    // 只處理大樂透檔案
    for (let i = 0; i < filteredFiles.length; i++) {
        // 上傳邏輯...
    }
}
```
