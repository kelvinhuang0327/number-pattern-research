### 1️⃣ 新增預測器模組

#### 1.1 社群智慧預測器 (`social_wisdom_predictor.py`)
**檔案位置**: `/models/social_wisdom_predictor.py`

**核心功能**:
- 避開大眾最常選擇的號碼（生日1-31、幸運數字7/8/9）
- 計算每個號碼的「非熱門分數」
- 提供3種策略：激進冷門、平衡策略、保守策略
- 可生成8注多樣化號碼

**主要類別**:
- `SocialWisdomPredictor`: 主預測器類
  - `predict()`: 預測單注號碼
  - `predict_with_balance()`: 平衡冷熱號碼
  - `generate_8_bets()`: 生成8注
  - `analyze_popularity()`: 分析號碼熱門度

**關鍵參數**:
- 生日號碼 (1-31) 權重降至 0.3-0.5
- 幸運數字權重降至 0.35-0.7
- 高號區 (42-49) 權重提升至 1.8

#### 1.2 量子隨機預測器 (`quantum_random_predictor.py`)
**檔案位置**: `/models/quantum_random_predictor.py`

**核心功能**:
- 使用澳洲國立大學 (ANU) 量子隨機數 API
- Fallback 使用 Python `secrets` 模組（密碼學級隨機）
- 可設定和值約束 (min_sum, max_sum)
- 隨機性品質測試（Chi-square test）

**主要類別**:
- `QuantumRandomPredictor`: 主預測器類
  - `predict()`: 基本隨機預測
  - `predict_with_constraints()`: 帶約束的隨機預測
  - `generate_8_bets()`: 生成8注（可確保多樣性）
  - `benchmark_randomness()`: 測試隨機性品質

**隨機源**:
- 優先：ANU QRNG API (量子物理真隨機)
- 備用：Python `secrets` (密碼學隨機)

#### 1.3 異常檢測預測器 (`anomaly_predictor.py`)
**檔案位置**: `/models/anomaly_predictor.py`

**核心功能**:
- 訓練模型識別「正常」號碼組合
- 預測「異常」組合（反向思維）
- 使用 Isolation Forest (如果有 sklearn) 或簡化版
- 提取12維特徵向量

**主要類別**:
- `AnomalyPredictor`: 主預測器類
  - `fit()`: 訓練異常檢測模型
  - `predict()`: 預測最異常的組合
  - `generate_8_bets()`: 生成8注異常組合
  - `analyze_combination()`: 分析號碼異常程度

**特徵維度** (12維):
1. 和值
2. 奇數個數
3. 大號個數 (>24)
4-5. 最小值、最大值
6. 間隔變異數
7. 連號數量
8-12. 五區分佈 (1-10, 11-20, 21-30, 31-40, 41-49)

#### 1.4 元學習預測器 (`meta_predictor.py`)
**檔案位置**: `/models/meta_predictor.py`

**核心功能**:
- 整合所有預測方法的結果
- 使用加權投票選擇最佳號碼
- 支援情境自適應權重調整
- 生成多樣化的N注號碼

**主要類別**:
- `MetaPredictor`: 主預測器類
  - `predict_with_ensemble()`: 加權集成預測
  - `predict_diversified()`: 生成多樣化N注
  - `analyze_bet_quality()`: 分析號碼質量
  - `adaptive_weights()`: 情境自適應權重

**預設權重配置**:
```python
default_weights = {
    'entropy': 0.30,      # 熵驅動 Transformer
    'deviation': 0.25,    # 偏差分析
    'social': 0.20,       # 社群智慧
    'anomaly': 0.15,      # 異常檢測
    'quantum': 0.10,      # 量子隨機
}
```

## 🎯 核心創新點

### 1. 社群智慧（避開熱門號碼）
**理論基礎**: 中獎機率相同，但獨得獎金期望值不同

**實施策略**:
- 識別熱門號碼（生日、幸運數字）
- 降低熱門號碼權重 (0.3-0.7)
- 提升冷門號碼權重 (1.5-1.8)

**預期效果**:
- 中獎時獨得獎金機率提升 50%+
- 避開共識陷阱

### 2. 量子隨機（真隨機基準）
**理論基礎**: 大樂透是真隨機，對比驗證其他方法

**實施策略**:
- 使用 ANU 量子隨機數 API
- Fallback 使用密碼學隨機
- 可設定和值約束

**預期效果**:
- 作為 Baseline 對比
- 驗證其他方法是否優於隨機

### 3. 異常檢測（反向選號）
**理論基礎**: 傳統方法預測「正常」→ 反向選「異常」

**實施策略**:
- 訓練模型識別正常組合
- 使用 Isolation Forest 檢測異常
- 選擇最異常的組合

**預期效果**:
- 突破傳統方法的共識陷阱
- 提供差異化預測

### 4. 元學習集成（整合所有方法）
**理論基礎**: 不依賴單一方法，加權投票

**實施策略**:
- 整合5大創新方法
- 使用預設權重配置
- 支援情境自適應調整

**預期效果**:
- 綜合各方法優勢
- 平衡創新與穩健
