### 2. 評估過程詳解

```python
async def _evaluate_config(config, train_set, val_set, ...):
    """
    評估配置的適應度（在驗證集上的成功率）
    """
    success_count = 0
    
    # 🔄 對驗證集中的每一期進行預測
    for target in val_set:
        # 1️⃣ 使用訓練集和配置預測號碼
        predicted = self._predict_with_config(
            config, train_set, pick_count, min_num, max_num
        )
        
        # 2️⃣ 計算命中數
        hits = len(set(target['numbers']) & set(predicted))
        
        # 3️⃣ 判斷是否成功（命中 ≥3）
        if hits >= 3:
            success_count += 1
    
    # 4️⃣ 計算適應度 = 成功率
    fitness = success_count / len(val_set)
    
    return fitness  # 例如：0.2400
```
