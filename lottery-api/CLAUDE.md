### 🚫 GNN 和深度強化學習可行性評估

**不推薦理由**:
1. **數據量不足**: 大樂透僅2000+期，GNN理想需要5000+期
2. **降級方案失敗**: 圖中心性方法僅1.33%（比單注2.67%還差）
3. **彩票隨機性**: 深度學習容易過擬合，實際效果可能更差
4. **時間成本**: GNN完整訓練需1-2週，成功率<10%
5. **強化學習不適用**: 彩票不是序列決策問題，每期獨立

**實際預期**:
- GNN完整訓練: 3-4%（可能不如統計方法）
- 深度強化學習: 2-4%或訓練失敗
