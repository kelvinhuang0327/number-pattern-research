# 🚀 智能自動學習系統 - 快速參考

## 一鍵使用

### 步驟 1: 啟動後端
```bash
cd lottery_api
python app.py
# ✅ 服務運行在 http://localhost:5001
```

### 步驟 2: 打開前端
```bash
# 在瀏覽器訪問
http://localhost:8000
```

### 步驟 3: 使用功能

#### 🎯 智能策略評估（找出最佳預測方法）
```
1. 選擇彩券類型（例如：大樂透）
2. 點擊「同步數據到後端」
3. 選擇測試集比例（推薦 20%）
4. 點擊「開始評估策略」
5. 等待 30-60 秒
6. 查看最佳策略和所有策略排名
```

#### ⚡ 參數優化（遺傳算法）
```
1. 設置遺傳代數（推薦 20）
2. 設置種群大小（推薦 30）
3. 點擊「開始優化」
4. 查看進化趨勢圖
```

#### ⏰ 自動排程（每天自動優化）
```
1. 設置執行時間（推薦 02:00）
2. 點擊「啟動排程」
3. 系統每天自動執行
```


## 🐛 常見問題

**Q: 評估失敗？**
```
1. 確認後端運行 (http://localhost:5001/health)
2. 確認已同步數據（點擊「同步數據到後端」）
3. 確認數據量足夠（至少 40 期）
```

**Q: 評估很慢？**
```
- 正常情況：30-60 秒
- 數據量大：可能需要 1-2 分鐘
- 選擇較小的測試集比例（10%）加速
```

**Q: 最佳策略會變嗎？**
```
會！原因：
- 數據量增加
- 歷史趨勢改變
- 彩票規則調整

建議每週重新評估一次
```
