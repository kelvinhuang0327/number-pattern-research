## Fusion Types

### weighted_rank
Rank each signal independently (1..39), then compute weighted sum of ranks.
```
combined[n] = Σ w_i × (39 - rank_i(n))
```

### score_blend
Normalize each signal to [0,1], then weighted sum.
```
combined[n] = Σ w_i × (score_i(n) - min_i) / (max_i - min_i)
```

### voting
Each signal votes for its top-10 numbers with positional weight.
```
combined[n] = Σ w_i × (10 - rank_in_top10_i(n))  for n in top-10 of signal i
```

### rank_product
Product of weighted inverse ranks (geometric mean variant).
```
combined[n] = 1 / Π rank_i(n)^w_i
```
