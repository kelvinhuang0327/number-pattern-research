## 策略簡介

### fourier_rhythm_3bet（主力）

- **方法**：傅立葉週期分析，識別號碼出現的頻率節律
- **特性**：MidFreq 中頻段偏好（避開極高頻/極低頻號碼）
- **300p Edge**：+3.16%（相對隨機基準）
- **Sharpe**：0.090
- **驗證**：三窗口全 PASS，perm test ✅

### pp3_freqort_4bet

- **方法**：PP3 框架 × 頻率正交選號
- **特性**：四注覆蓋，正交化設計減少重複
- **300p Edge**：+3.40%
- **Sharpe**：0.088
- **驗證**：perm p=0.000，McNemar net=+65
