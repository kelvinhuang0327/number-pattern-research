## 瀏覽器手動測試步驟

### 1. 準備測試數據

上傳不同彩券類型的 CSV 檔案：
- `大樂透_2024.csv` (1-49, 6個號碼)
- `三星彩_2024.csv` (0-9, 3個號碼)
- `今彩539_2024.csv` (1-39, 5個號碼)
- `四星彩_2024.csv` (0-9, 4個號碼)

### 2. 測試彩券類型篩選

1. 打開應用程式 (`index.html`)
2. 上傳包含多種彩券類型的 CSV 檔案
3. 使用「彩券類型」下拉選單切換不同類型
4. **驗證**：
   - 數據摘要顯示正確的記錄數
   - 歷史記錄只顯示選定類型的數據
   - 圖表 X 軸範圍正確（0-9 或 1-49 等）

### 3. 測試預測功能

對每個彩券類型，測試所有 14 個策略：

#### 三星彩 (0-9, pick 3)
```
策略列表：
✓ frequency (頻率分析)
✓ trend (趨勢分析)
✓ hot_cold (冷熱號混合)
✓ collaborative_hybrid (協作混合)
✓ deviation (偏差追蹤)
✓ odd_even (奇偶平衡)
✓ zone_balance (區間平衡)
✓ sum_range (和值範圍)
✓ bayesian (貝葉斯)

✓ number_pairs (連號配對)
✓ statistical (多維統計)

預期結果：3 個號碼，範圍 0-9
```

### 4. 驗證檢查清單

對每個預測結果，檢查：

- [ ] 號碼數量正確（三星彩=3, 四星彩=4, 今彩539=5, 大樂透=6）
- [ ] 號碼範圍正確（三星彩/四星彩=0-9, 今彩539=1-39, 大樂透=1-49）
- [ ] 沒有重複號碼
- [ ] 號碼為整數
- [ ] 顯示信心度百分比
- [ ] 顯示策略名稱

### 5. 測試圖表

切換不同彩券類型，驗證：

- [ ] **頻率圖表**: X 軸範圍正確
- [ ] **遺漏值圖表**: X 軸範圍正確
- [ ] **分佈圖表**: 顯示正確的號碼範圍
- [ ] **冷熱號碼**: 顯示正確範圍的號碼

### 6. 測試統計數據

驗證統計摘要：

- [ ] 總期數正確
- [ ] 最新期號正確
- [ ] 日期範圍正確
- [ ] 彩券類型標籤正確
