# 🚀 後端優化預測 - 快速開始指南

## 什麼是後端優化預測？

後端優化預測是一個**全新的預測選項**，它直接使用後端遺傳算法優化過的參數進行預測，無需前端重新計算。

### 🎯 核心優勢
- ✅ **成功率高**: 接近 10% 的成功率（相比傳統方法的 3-5%）
- ✅ **速度快**: 直接使用已優化的參數，無需重新計算
- ✅ **簡單易用**: 只需選擇一個選項即可

---

## 📋 使用前準備（三步驟）

### 步驟 1: 啟動後端服務

打開終端，執行：

```bash
cd lottery-api
python app.py
```

看到以下信息表示成功：
```
INFO:     Uvicorn running on http://0.0.0.0:5001
```

### 步驟 2: 同步數據到後端

1. 在前端網頁上傳或載入彩票數據
2. 切換到「🤖 自動學習」頁面
3. 點擊「📤 同步數據到後端」按鈕
4. 看到成功提示

### 步驟 3: 執行一次優化（首次使用）

在「自動學習」頁面：
1. 設置遺傳代數：20（建議值）
2. 設置種群大小：30（建議值）
3. 點擊「🚀 開始優化」
4. 等待優化完成（約 2-5 分鐘）

## 🎮 如何使用

### 在「智能預測」頁面

1. 選擇預測方法：**🚀 後端優化預測 (10%成功率)**
2. 點擊「🔮 開始預測」
3. 查看預測結果

### 在「模擬測試」頁面

1. 選擇年度月份（例如：2025-01）
2. 選擇預測方法：**🚀 後端優化預測 (10%成功率)**
3. 點擊「🧪 開始模擬」
4. 查看模擬結果和成功率

## 🔧 常見問題

### Q1: 提示「無法連接到後端服務器」？
**A**: 後端服務未運行，請執行：
```bash
cd lottery-api
python app.py
```

### Q2: 提示「尚未執行自動優化」？
**A**: 需要先執行一次優化：
1. 到「自動學習」頁面
2. 點擊「開始優化」
3. 等待完成

### Q3: 提示「後端沒有數據」？
**A**: 需要同步數據：
1. 到「自動學習」頁面
2. 點擊「同步數據到後端」

### Q4: 如何提高成功率？
**A**: 
- 定期執行優化（建議每週一次）
- 或啟用自動排程（每天自動優化）
- 確保數據是最新的

## 💡 最佳實踐

1. **啟用自動排程**: 設置每天凌晨 2:00 自動優化
2. **定期更新數據**: 有新開獎數據時，記得同步到後端
3. **多方法對比**: 可以同時使用多種方法，對比結果
4. **記錄結果**: 追蹤不同方法的實際成功率
