# 3星彩預測之實體隨機與物理特性分析
**身分：隨機過程數學家 + 物理模擬專家**

如果數字序列在統計上表現出非隨機性，通常源於底层抽獎系統的物理缺陷。

## 1. 物理偏差來源
- **球體磨損與重量差異**：即使是 0.1 克的差異，在高速氣流抽獎機中也會改變其懸浮軌跡。
- **空氣動力學偏差**：抽獎箱內的氣流循環若不完全對稱，會導致某些位置的球更易進入吸管。
- **開獎頻率與維護周期**：抽獎機的精密零件（如攪拌翼）會隨使用時間產生磨損，導致特有的「機器指紋」。

## 2. 檢測方法：物理特徵挖掘
我們將開獎數據轉化為以下物理代理指標：
- **位置偏好 (Positional Bias)**：球從特定管道排出的頻率。
- **溫度/濕度相關性**：某些材質在不同濕度下的摩擦力不同。
- **區間集群 (Clustering)**：數字是否傾向於成對出現（物理上的相近性）。

## 3. 建模方法：狀態空間模型 (State Space Model)
假設抽獎機存在一個「隱藏狀態」 $S_t$，其物理衰減遵循某種轉移過程。
$$S_{t+1} = f(S_t) + \epsilon$$
我們透過觀察開獎號碼 $y_t$ 來過濾並估計隱藏狀態 $S_t$，從而預測短期的趨勢漂移。

## 4. 驗證
若歷史數據中出現 **持久性的自相關 (Long-term Autocorrelation)**，則物理偏差假設成立，機率預測將具有實際盈利空間。
