# 回測驗證指令

回測已驗證策略，計算 Edge vs 隨機基準。

## 使用方式

```
/backtest 大樂透             # 3注 Triple Strike, 1000期
/backtest 大樂透 2            # 2注 P0 回聲, 1000期
/backtest 威力彩              # 2注 Fourier Rhythm, 1000期
/backtest 威力彩 3            # 3注 Power Precision, 1000期
/backtest 大樂透 --three-tier # 150/500/1500 三階驗證
/backtest 威力彩 --periods 500 # 自訂期數
/backtest all                 # 全部彩票默認注數
```

## 執行指令

收到此命令後，執行以下腳本：

```bash
python3 tools/quick_backtest.py $ARGUMENTS
```

## 輸出格式

### 單次回測
- 命中分布 (Match-4 ~ Match-0) + 直方圖
- M3+ 命中數/期數 (百分比)
- 隨機基準 (N注)
- **Edge = M3+率 - 基準** → PASS/FAIL

### 三階驗證 (--three-tier)
- 150/500/1500 期三窗口 Edge
- 趨勢分類: STABLE / MODERATE_DECAY / LATE_BLOOMER / SHORT_MOMENTUM / INEFFECTIVE

## 參數說明

- **$1** (必填): 彩票類型 (大樂透/威力彩/all)
- **$2** (選填): 注數 (默認: 大樂透=3, 威力彩=2)
- **--periods / -p**: 回測期數 (默認 1000)
- **--three-tier / -t**: 執行 150/500/1500 三階驗證
- **--seed / -s**: 隨機種子 (默認 42)
