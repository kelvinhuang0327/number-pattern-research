# 台湾大乐透回本策略分析

## 💰 奖金结构（每注100元）

| 奖项 | 中奖条件 | 奖金 |
|------|---------|------|
| 头奖 | 6+特别号 | 超高 |
| 二奖 | 6+0 | 数百万 |
| 三奖 | 5+特别号 | 数万 |
| 四奖 | 5+0 | 数千 |
| 五奖 | 4+特别号 | 数百 |
| 六奖 | 4+0 | ~400元 |
| 七奖 | 3+特别号 | ~400元 |
| **八奖** | **3+0** | **200元** ✅ |
| 九奖 | 2+特别号 | 100元 |

## 🎯 回本目标分析

**成本**: 买两注 = 200元

**回本方案**：
1. 一注中3个普通号 = 200元 ✅ 回本
2. 两注各中2个+特别号 = 200元 ✅ 回本
3. 一注中2个+特别号 + 一注中2个普通号 = 100元 ⚠️ 亏100
4. 一注中4个 = 400元 🎉 赚200

## 📊 概率计算

### 单注概率
- 中3个普通号：~1.77%
- 中2个+特别号：~0.97%
- 中4个：~0.097%

### 两注策略概率
假设两注独立：
- 至少一注中3个：~3.5%
- 至少一注中2+特：~1.9%
- **至少回本**：~5-6%

## 💡 优化策略

### 策略1：独立预测
- 两注使用不同算法
- 增加号码覆盖范围
- 提高至少一注中奖概率

### 策略2：互补策略  
- 第一注：高频号码
- 第二注：温号+冷号
- 最大化覆盖范围

### 策略3：特别号优化
- 单独优化特别号预测
- 提高"2+特"的概率
- 两注可以用同一特别号
