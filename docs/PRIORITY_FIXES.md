| `LOTTO_49` | `49_LOTTO` | `49_LOTTO` |
| `LOTTO_38` | `38_LOTTO` | `38_LOTTO` |

**修正文件**:

## ⚠️ 重要修正（本週內）

### 2. 修正樂合彩系列配置

**問題**: 39/49/38樂合彩的 `pickCount` 配置錯誤

**當前配置問題**:
```json
"39_LOTTO": {
    "pickCount": 5  // ❌ 錯誤
}
```

**正確配置應為**:
```json
"39_LOTTO": {
    "playModes": {
        "二合": { "pickCount": 2, "matchRequired": 2 },
        "三合": { "pickCount": 3, "matchRequired": 3 },
        "四合": { "pickCount": 4, "matchRequired": 4 }
    },
    "dependsOn": "DAILY_539",
    "isSubGame": true
}
```

**影響**:
- 無法正確處理多玩法
- 預測結果不準確

**預計時間**: 2-3 小時
